## Was ist mintoryx?

mintoryx ist eine Web-Agentur die professionelle, moderne Websites für kleine und mittlere Unternehmen baut — erschwinglich, sauber und ohne Baukastensystem.

## Features

- **Parallax Scroll Hero** — Bild öffnet sich wie ein Fenster beim Scrollen
- **Orbit Animation** — Skills kreisen animiert um ein Profilbild
- **Expand Cards** — Interaktive Prozesskarten mit Hover-Effekt
- **Bento Grid** — Modernes Layout für die "Warum"-Section
- **Projekt-Modal** — Kontaktformular das direkt ein Mail vorbereitet
- **Canvas Pfeil** — Dynamischer Maus-Pfeil zeigt auf den CTA-Button
- **Interaktives Nav** — Pill-Navigation mit Scroll-Tracking

## Stack

Kein Framework. Kein Template. Reines Handwerk.

```
HTML · CSS · Vanilla JS
```
